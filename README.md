# 🐔 AyamTech — Broiler Chicken Farm Management System

A database-driven decision support system for broiler 
chicken farm management built as a Final Year Project 
at Universiti Teknikal Malaysia Melaka (UTeM).

## 🛠️ Tech Stack
- **Database:** PostgreSQL
- **Backend:** Node.js, Express.js
- **Frontend:** React.js, JavaScript
- **Tools:** DBeaver, pgAdmin, GitHub

## 📌 Features
- Production cycle & batch management
- Feed, medication & mortality tracking
- Sales & financial reporting dashboard
- Decision support analytics
- Multi-role access (Owner, Worker, Driver, Customer)
